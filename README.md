# OSOMNImedia-tools

$MFI PROJECT uploads and then displays the uploaded data.

Features
--------

- Upload Masterfile
- Upload MFI
- Upload MFI Health
- Redo uploaded Masterfile
- View Brands, Vendors, All
- Export Total FBA Value
- Export Potential Trouble SKUS
- Export Brands, Vendors, All
- Log-in system
  > Admin rights
